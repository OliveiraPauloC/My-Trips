# Projeto My Trips

### Um projeto pessoal de um mapa interativo utilizando Leaflet e gerenciando os dados dos pontos pelo GraphCMS.

## Tecnologias utilizadas:
- React
- NextJS
- Typescript
- Styled-components
- Hooks
- Leaflet
- GraphQL
- GraphCMS

## Link Vercel 
### Visite: [MyTrips](https://my-trips-devpaulo.vercel.app/)
